# Визуализация алгоритмов сортировки.
Внимание!!! Поддерживаются только современные браузеры.

## Что реализовано
    -Сортировка пузырьком
    -Сортировка вставкой
    -Сортировка слиянием
    -Сортировка выбором
    -Быстрая сортировка


# Установка
```sh
npm install
npm run serve
```
После чего приложение будет доступно по http://localhost:9080/
